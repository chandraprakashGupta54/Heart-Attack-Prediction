# 🫀 Heart Attack Prediction App

An AI-powered web application that predicts the risk of heart disease using Machine Learning.

Built with:
- Python
- Scikit-learn
- Streamlit
- Logistic Regression

---

## 🚀 Live App

👉https://heartattackprediction5038.streamlit.app/

---

## 📌 Project Overview

This application predicts whether a person is at **High Risk** or **Low Risk** of heart disease based on medical parameters such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol Level
- Fasting Blood Sugar
- Resting ECG Results
- Maximum Heart Rate
- Exercise-Induced Angina
- ST Depression (Oldpeak)
- ST Slope

The model was trained using Logistic Regression and deployed using Streamlit Cloud.

---

## 🧠 Machine Learning Workflow

1. Data Cleaning
2. One-Hot Encoding for categorical features
3. Feature Scaling using StandardScaler
4. Train-Test Split
5. Model Training (Logistic Regression)
6. Model Saving using Joblib
7. Web Deployment using Streamlit
